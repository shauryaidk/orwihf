About Syntax & Spices

Your go-to guide for dorm-friendly cooking.

Welcome to Syntax & Spices! This website was built from scratch using pure HTML and CSS as part of a Web Development assignment. The goal? To prove that you don't need complex code to create a beautiful, responsive, and mouth-watering recipe directory.


Every recipe featured here is hand-picked for busy college students. Whether you are prepping for the JEE, pulling an all-nighter for an exam, or just need a quick bite between classes, these recipes use minimal ingredients and simple steps to get you fed fast.
